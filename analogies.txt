response / request methods

response:
.send() - Sends a basic string or HTML.
.json() - Sends a JavaScript object as JSON.
.status() -  Sets the HTTP code (200, 404, 500).
.redirect() - Forces the browser to go to a new URL.
.sendfile() - Sends an actual file (like an image or HTML file).

request:
req.body()
req.params()
req.query()
req.headers()
